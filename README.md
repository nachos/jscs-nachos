## Installation
``` bash
$ [sudo] npm install jscs-nachos --save-dev
```

## Usage
### Load preset
In your .jscsrc file, add this:
``` json
{
  "preset": "nachos"
}
```

## Run Tests
``` bash
$ npm test
```
